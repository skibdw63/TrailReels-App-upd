# TrailReels Updated

This is the normal GitHub + Netlify version.

## Netlify settings

Build command:

```txt
npm run build
```

Publish directory:

```txt
dist
```

## Important GitHub rule

When you upload to GitHub, the first page of the repo must show:

```txt
package.json
index.html
src
netlify.toml
manifest.webmanifest
icon.svg
README.md
```

Do NOT upload the folder inside another folder.

## What works in this preview

- Feed, Memories, Live, Premiere, Reelers, Profile, Owner Panel
- Buttons switch pages
- Play/pause animation
- Report button sends reports to Owner Panel
- Gift/Super Echo system
- Premiere chat
- Badge creation
- Upload popup
- Owner crown and creator badge icons
